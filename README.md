# 投資判断AIノート

GitHub Pages 公開用の Vite + React アプリです。

## 重要

このZIPの中身を、GitHubリポジトリ `kabusupport` の**直下**に置いてください。

正しい配置:

```text
kabusupport/
├─ .github/
├─ package.json
├─ index.html
├─ vite.config.js
├─ .gitignore
└─ src/
```

間違い:

```text
kabusupport/
└─ kabusupport/
   ├─ package.json
   └─ src/
```
